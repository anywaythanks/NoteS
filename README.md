# NoteS
## Startup
Предварительно собрать через `npm` фронт.
```zsh
cd Client
npm install
```
Собрать asp или взять бинарники из релиза.
```zsh
dotnet build

```
Запустить докер:
```zsh
docker compose up
```
И бэк на шарпе.
