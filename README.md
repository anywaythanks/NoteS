# NoteS
Невероятно важный и оооочень интересный проект для вузека.

## Startup
Запустить докер:
```zsh
docker compose
```
Предварительно собрать через `npm` фронт.
```zsh
cd Client
npm install
```
Запустить шапровое чудо. Проще взять бинарники из релиза. Если он будет, ха-ха.
